## Lab Setup

### Virtual Machines
- Ubuntu 24.04 LTS (IP: 10.0.2.15) - Target/Attacker machine
- Kali Linux 2026.1 (IP: 10.0.2.3) - SOC machine running Python agent

### Network
- VirtualBox NAT Network (labnet)
- Both VMs on 10.0.2.0/24 subnet

### Requirements
- VirtualBox 7.2+
- 8GB RAM minimum
- Python 3, tshark, requests library
- Airia AI account and API key
